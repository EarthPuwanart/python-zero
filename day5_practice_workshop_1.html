Built-in Functions ที่ใช้บ่อย

Functionใช้ทำอะไรตัวอย่าง
abs(x)ค่าสัมบูรณ์abs(-5) → 5
max(...)ค่ามากที่สุดmax(3, 7, 2) → 7
min(...)ค่าน้อยที่สุดmin([3, 7, 2]) → 2
sum(list)ผลรวมของข้อมูลใน listsum([10, 20, 30]) → 60
len(x)จำนวนสมาชิก/ความยาวข้อความlen("data") → 4
round(x, n)ปัดทศนิยมround(3.14159, 2) → 3.14

💡 ให้ฝึกใช้กับ list ของตัวเลข เพราะจะเชื่อมกับโจทย์สถิติพื้นฐานได้ทันที

🎯 เคล็ดลับการทำโจทย์

  • อ่านโจทย์ให้ครบ — ทำความเข้าใจ input/output ก่อน
  • แบ่งปัญหาเป็นขั้นตอน — อย่ารีบเขียนโค้ดทันที
  • เขียนทีละส่วน — ทดสอบทุกขั้น อย่ารอเขียนเสร็จแล้วค่อยรัน
  • ใช้ print() debug — เช็คค่าตัวแปรระหว่างทาง
  • อ่าน Error message — Python บอกว่าผิดตรงไหน!
