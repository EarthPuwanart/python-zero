REVIEW

หัวข้อที่ต้องทบทวน

Variables

ชื่อ, type, casting, กฎการตั้งชื่อ

Operators

+,-,*,/,//,%,**, ลำดับ, assignment

String

Collections

List methods, Dict access/methods

Conditionals

if, elif, else, การไล่ลำดับเงื่อนไข

Loops

Functions

Built-ins

abs, max, min, sum, len, round

TIPS

💡 เคล็ดลับการทำข้อสอบ

  • ทำข้อง่ายก่อน — อย่าติดข้อยากนานเกินไป
  • Trace โค้ดด้วยมือ — เขียนค่าตัวแปรทุกบรรทัด
  • ระวัง index — เริ่มที่ 0 เสมอ
  • ระวัง type — input() ได้ str, / ได้ float, // ได้ int
  • เช็คเงื่อนไข — if-elif ทำแค่เงื่อนไขแรกที่ True
  • อ่านให้ครบ — ดู = กับ == ให้ดี!
