Variables & Data Types

กฎการตั้งชื่อตัวแปร

  • ต้องประกอบด้วย ตัวอักษรภาษาอังกฤษ (a-z, A-Z), ตัวเลข (0-9) หรือขีดล่าง (_) เท่านั้น
  • ห้าม ขึ้นต้นด้วยตัวเลข
  • ห้าม มีช่องว่าง (เว้นวรรค) หรือใช้เครื่องหมายพิเศษอื่นๆ (เช่น @, #, -)
  • ห้าม ตั้งชื่อซ้ำกับคำสงวนของภาษา Python (เช่น if, for, class, True, False)
  • ตัวพิมพ์เล็กและพิมพ์ใหญ่ถือเป็นคนละตัวแปร (Case-Sensitive)

Input & Conversion

✂️ split() — รับหลายค่าในบรรทัดเดียว

แยกค่าด้วย split()

# รับชื่อและนามสกุล
data = input("Enter name: ")
first, last = data.split()
print("First:", first)
print("Last:", last)
Enter name: Alice Smith First: Alice Last: Smith

แยกด้วยตัวคั่นอื่น

# แยกด้วย comma
data = "Alice,20,3.75"
name, age, gpa = data.split(",")

print(name)
print(age) # 20 (ยังเป็นข้อความ!)
print(gpa) # 3.75 (ยังเป็นข้อความ!)

💡 split() default แยกด้วย space — ระบุตัวคั่นอื่นได้ เช่น split(",")

Input & Conversion

🔢 แปลงค่าหลัง split() ทีละตัว

ปัญหา: split() ได้ str ทั้งหมด

# ❌ ใช้ split() อย่างเดียว
a, b = input().split()
print(a + b)
# Input: 10 20
"1020" ← เชื่อมข้อความ!

วิธีแก้: แปลงด้วย int() หลัง split()

# ✅ แยกก่อน แล้วค่อยแปลงทีละตัว
a, b = input().split()
a = int(a)
b = int(b)
print(a + b)
# Input: 10 20
30 ← ถูกต้อง!

💡 สำหรับผู้เริ่มต้น วิธีนี้อ่านง่ายและเห็นชัดว่า input() ได้ข้อความก่อน แล้วเราค่อยแปลงเป็นตัวเลข
